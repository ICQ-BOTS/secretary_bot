# Бот-секретарь

[Бот-секретарь](https://icq.im/secretary_bot)

Старт:
1. Установка всех зависимостей 
```bash
pip3 install -r requirements.txt
```

2. Инициализация базы данных
```bash
python3 database.py
```

3. Вставляем токен в config.py

4. Запуск бота!
```bash
python3 secretary_bot.py
```
